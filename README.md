# PRODIGY_ML_03

## Task
Implement a Support Vector Machine (SVM) to classify images of cats and dogs.

## Dataset
Dogs vs Cats Dataset from Kaggle

## Preprocessing
- Resize images to 64x64
- Convert images to grayscale
- Flatten image pixels

## Tools Used
- Python
- OpenCV
- Scikit-learn
- NumPy
- Matplotlib

## Output Files
- output.txt → stores model results
- output.png → terminal output screenshot
- sample_predictions.png → predicted sample images

## Result
Successfully classified cats and dogs using SVM.
